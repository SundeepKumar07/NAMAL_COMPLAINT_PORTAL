This portal provides role-based access for different types of users:
- **Admin:** Manages all complaints, assigns roles, and monitors progress.
- **Faculty:** Can submit, track, and manage department-related complaints.
- **Non-Teaching Staff:** Can raise and respond to maintenance or office-related issues.
- **Students:** Can submit complaints regarding academic, hostel, or campus facilities.

Each role has specific permissions and dashboard views to ensure smooth and secure handling of complaints.
